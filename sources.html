Sources

Cette page présente les données et outils utilisés pour afficher les radars fixes en France sur la carte.

Données utilisées:

Les radars sont issus du jeu de données « Liste des radars fixes en France » publié sur data.gouv. Le jeu de données recense l'emplacement des radars fixes, leur typologie ainsi que d'autres champs descriptifs.

  • Source: data.gouv (ANTAI)
  • Type: radars fixes
  • Format utilisé: JSON / GeoJSON
  • Mise à jour: variable selon publication

Lien du dataset: Liste des radars fixes en France

Technologies utilisées:

  • Leaflet pour l'affichage cartographique
  • Fetch API pour la récupération des données
  • JavaScript pour le traitement
  • CSS pour la mise en forme
